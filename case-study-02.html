Client Snapshot

Practice Type

Outpatient physical therapy clinic

Size

Solo PT · 2 admin staff · ~22 patients/day

EHR

WebPT

Before

PT dictating/typing SOAP notes after hours. Insurance checks done manually by admin before each new patient visit.

Stack

n8n (cloud) · Whisper AI (voice transcription) · OpenAI GPT-4o · Availity API · WebPT webhook · Gmail

The Problem

  • Therapist spending 2+ hours nightly on SOAP note documentation

  • Insurance verification taking 45 minutes per new patient

  • Admin bottleneck delaying new patient starts by days

  • No capacity to grow the practice without hiring additional staff

The Solution

  • Voice memo transcribed by Whisper AI on session end trigger

  • GPT-4o drafts structured SOAP note — clinical review under 5 minutes

  • Insurance eligibility checked automatically on intake form submission

  • New patient onboarding reduced from 60+ minutes to under 15 minutes

Automation Sequence

1

Trigger · WebPT

Session ends

WebPT fires a webhook when the therapist marks the session complete, passing session ID and patient context.

2

Transcription · Whisper AI

Voice memo transcribed

The therapist's voice memo is sent to Whisper AI for transcription. Audio processed in memory — never stored.

3

Generation · OpenAI GPT-4o

SOAP note drafted

GPT-4o generates a structured, clinically accurate SOAP note from the transcript using a PT-specific prompt template.

4

Review · WebPT

PT reviews draft in EHR

Draft is written directly into WebPT. Therapist reviews, edits if needed, and signs off — average review time: 4 minutes.

5

Trigger · New Patient Intake

Insurance check auto-runs

When intake form is submitted, the workflow automatically queries Availity for eligibility and benefits information.

6

Delivery · Gmail

Eligibility summary emailed to admin

A formatted eligibility and benefits summary is emailed to admin before the patient's first appointment — no manual lookup needed.

Results

80%

Documentation Time Cut

Compliance & Privacy

Voice memos are processed in memory only and never stored. SOAP drafts are written directly to the clinic's HIPAA-compliant EHR. Insurance data is queried via payer APIs with no PHI stored externally. All workflow credentials are encrypted at rest.
